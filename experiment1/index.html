Experiment 1 - Superhero Prompt Generator

Description

Generate prompts of a concerned citizen asking for help to stop a supervillain, with randomized word inputs.

Technical

A basic template in a text box has a few words that can be randomly selected from an array of strings. Each time the prompt is presented, or when the reroll button is pressed, new random prompts are generated.

Reflection

This is a simple experiment just to test out how Glitch works, so I did not really do much of anything worth reflecting.
